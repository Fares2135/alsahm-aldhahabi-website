# موقع مصنع السهم الذهبي — التصميم الجديد

موقع ثابت (HTML/CSS/JS فقط، بدون أطر عمل أو خطوات بناء). افتح `index.html` مباشرة في المتصفح، أو ارفع مجلد `redesign/` كاملًا إلى أي استضافة.

## الصفحات

| الملف | المحتوى |
|---|---|
| `index.html` | الرئيسية: البطل، المزايا، من نحن، الأرقام، فئات المنتجات، القيم، لماذا نحن، الشهادات، التصدير، الأسئلة الشائعة |
| `about.html` | من نحن + الرؤية والمهمة + القيم + النهج والجودة + رسالة الرئيس التنفيذي |
| `sectors.html` | القطاعات الستة (جملة، تجزئة، صناعي، تصدير، تعبئة، حكومي) + برنامج ولاء بلس |
| `products.html` | كامل المنتجات (36 منتجًا في 8 فئات) مع تنقّل سريع بين الفئات |
| `quality.html` | آلية التصنيع (9 مراحل) + المختبرات + الشهادات (ISO 9001/14001/45001، ساسو) |
| `export.html` | التصدير حول العالم: الخريطة، الأسواق، معايير التصدير |
| `investors.html` | نظرة عامة، الحوكمة، المراجعة الداخلية، المؤشرات المالية، معلومات الاتصال |
| `contact.html` | بيانات الاتصال + نموذج رسالة (يفتح بريد العميل — لا يوجد خادم) |

## نظام التصميم

**الألوان** (مستخرجة من الشعار) — معرفة كمتغيرات CSS في أول `assets/css/style.css`:

- كحلي العلامة: `--navy-700: #1D2B5E` (الأساسي)، تدرجات حتى `--navy-950: #0B1229` للخلفيات الداكنة
- ذهبي العلامة: `--gold-500: #C2A045` (الأزرار والتأكيد)، `--gold-400/600` للتفاعل
- نصوص: `--ink: #182038` للعناوين، `--body: #4C5570` للفقرات
- خلفيات: أبيض + `--bg-alt: #F5F7FB` للتناوب بين الأقسام

**الخطوط** (Google Fonts):
- العناوين: **Alexandria** (أوزان 500–800)
- النصوص: **IBM Plex Sans Arabic** (أوزان 400–700)

**الإيقاع**: حاوية 1180px، حشوة الأقسام `clamp(64px, 9vw, 112px)`، زوايا 14/22px، مقياس مسافات مبني على 8px.

**المكوّنات** (كلها في `style.css`): أزرار (ذهبي/كحلي/شفاف/محدد)، بطاقات بأيقونات، بطاقة منتج بشارة اللزوجة، خطوات مرقمة، بطاقات شهادات، أكورديون أسئلة (بدون JS — عنصر `<details>`)، شريط دعوة لاتخاذ إجراء، إحصائيات، رأس وذيل موحدان.

**التجاوب**: نقاط كسر 1040 / 900 / 620px — قائمة جوال منسدلة تحت 900px.

## الخريطة التفاعلية

`assets/js/worldmap.js` يبني خريطة عالم تفاعلية (SVG) داخل أي عنصر يحمل `data-worldmap` — مستخدمة في الرئيسية وصفحة التصدير:

- السعودية بالذهبي مع نبض على الرياض، وأقواس شحن متحركة نحو الأسواق.
- أفريقيا وآسيا (نطاق أسواق التصدير) بلون ذهبي شفاف، وتلميحات عربية عند التمرير (أسماء الدول عبر `Intl.DisplayNames` المدمجة في المتصفح — بلا أي بيانات إضافية).
- أزرار تصفية: كل الأسواق / السعودية / أفريقيا / آسيا والشرق الأوسط.
- مسارات الدول من خريطة amCharts worldLow (مجانية الاستخدام). لتعديل الأسواق حرّر القائمتين `AFRICA` و`ASIA` أو `ARC_TARGETS` أعلى الملف.

## ملاحظات

- المصدر: محتوى الموقع الأصلي محفوظ من أرشيف `site_archive/` (نسخة 12 يوليو 2026).
- صفحات المنتجات في الموقع الأصلي كانت تستخدم صورة واحدة مكررة لعلامة منافسة (CRONUS)، لذلك استُبدلت ببطاقات طباعية نظيفة بشارة اللزوجة. عند توفر صور حقيقية للعبوات، أضفها داخل `.pcard`.
- نموذج التواصل يفتح بريد العميل (mailto) لعدم وجود خادم؛ اربطه لاحقًا بخدمة نماذج أو API عند الاستضافة.
- ملف الشركة: `assets/docs/company-profile.pdf`.
